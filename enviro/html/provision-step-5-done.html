Your board is configured and ready to rock. All that's left to do is reset your board so that it can start gathering data.

Once reset your board will enter data gathering mode.


If you need to setup your board again.

If you want to use this setup tool again then you can get back into it by following these steps:

  • Hold down the POKE button – the one on the front with the ☞ icon.
  • Press and release the RESET button (on the back).
  • Keep POKE held down for three more seconds.
  • The activity ⚐ LED will start to flash.
  • The board will go into access point mode and appear as a WiFi network to connect to.
  • Connect to the board using your phone, laptop, stone circle, or tablet.


Advanced mode: manually editing your configuration.

All the questions we asked during setup were a convenient way to generate your boards' configuration file.

In future if you want to quickly tweak a setting without going through process again you can simply edit config.py using a tool like Thonny and then reset your board to apply the change.
